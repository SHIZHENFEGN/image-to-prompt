# Chrome 扩展开发计划：图生提示词

## TL;DR

> **快速摘要**: 开发一个 Chrome 浏览器插件，将网页图片转化为 AI 生图提示词
> 
> **输出**: 完整的 Chrome 扩展（MV3），包含 popup 界面、设置页面、后台 API 调用
> **预计工作量**: Medium
> **并行执行**: 是 - 3 个阶段

---

## 上下文

### 用户需求
- 开发 Chrome 浏览器插件
- 功能：把网页中的图片转化为 AI 生图提示词
- 使用国内 API（智谱 AI glm-4v）
- 提示词格式：Nano Banana Pro 格式
- UI 方式：扩展图标 + Popup

### 确认的需求
- API 服务商：智谱 AI（默认）
- 提示词格式：Nano Banana Pro 六要素结构
- 交互方式：点击扩展图标 → Popup 中选择图片 → 生成提示词

---

## 工作目标

### 核心目标
创建一个完整的 Chrome 浏览器扩展，能够：
1. 用户点击扩展图标打开 Popup
2. Popup 获取当前页面所有图片
3. 用户选择一张图片
4. 调用智谱 AI API 生成提示词
5. 显示结果并支持复制

### 必须实现的功能
- [ ] manifest.json（MV3 配置）
- [ ] popup.html/js/css（弹出界面）
- [ ] background.js（API 调用）
- [ ] content.js（图片提取）
- [ ] options.html/js/css（设置页面 - API Key 配置）

### 提示词格式（Nano Banana Pro）
```
[主体描述] 正在 [动作] 在 [场景]。[构图/镜头]。[光线/氛围]。[风格/媒介]。[质量参数]
```
