使用帮助

快速开始：安装扩展 → 配置 API → 悬停图片 → 点击生成

基本使用

1
安装扩展

在 Chrome 扩展管理页面开启"开发者模式"，点击"加载已解压的扩展程序"，选择本扩展文件夹。

2
配置 API

点击扩展图标打开设置页面，选择 API 服务商并填写 API Key。

3
生成提示词

在任何网页上，鼠标悬停在图片上，会显示"生成 AI 提示词"按钮。点击按钮即可生成。

常见问题

Q: 为什么点击按钮没反应？
A: 请检查：1) API Key 是否正确填写 2) 网络是否正常 3) API 是否有足够配额
Q: 生成的提示词是英文还是中文？
A: 生成的提示词为英文，适合 MJ、SD 等英文生图模型使用。
Q: 如何使用 Ollama 本地模型？
A: 1) 下载安装 Ollama 2) 运行 ollama pull llava 3) 在设置中选择 Ollama 并保存即可。
Q: 图片太大上传失败怎么办？
A: 尝试使用更小的图片，或选择支持更大图片的模型（如 glm-4v）。
Q: 可以商用生成的提示词吗？
A: 可以，生成的提示词无版权限制，可自由使用。
